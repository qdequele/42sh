# 42SH

42sh is an simple an light command line. This project is sourced by project of the 42School (@42Born2Code).

## Install

clone the directory.

`$ make`

## Test

`$ make test`

## Run as shell

move the executable on your path

`$ cp 42sh /usr/bin/`

set the access right

`$ chmod 755 /usr/bin/42sh`

open the shell

`$ 42sh`
